Welcome to the Overview Page

This page outlines the main elements of CodeBubbles to familiarize beginner users with the system.

CodeBubbles Major Elements

1. Workspace

Where you will view and edit your code.

2. Map

A map of your overall workspace.

3. Package Explorer

Shows an overview of the code.

4. Context Menu

The hub for most of Code Bubbles' functionality.

5. Editor Bubbles (or just Bubbles)

A specific code snippet, page, or function that you can view and edit.

6. Bubble Group

A grouping of bubbles, signified by the connected same-colored border that surrounds them.

7. Feature Buttons

Buttons that allow you to create new files, change options, get help,and more.

8. Task Menu

Menu that allows quick access to the refresh, build all, save all, and text search tasks. Opened and closed with F1.
