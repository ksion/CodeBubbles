Welcome to CodeBubbles

We’re stoked you want to try CodeBubbles!

Code Bubbles is a front end to Eclipse designed to simplify programming by making it easy for the programmer to define and use working sets. Working sets here consist of the group of functions, documentation, notes, and other information that a programmer typically needs to accomplish a programming task. Some examples include: adding a feature, fixing a bug, adding code documentation.

Drag navigation allows users to easily map out code bubbles in a sequential, readable order.

Code Navigation is completed with ease through the expansion of a single CodeBubble.

CodeBubbles's interface allows for the movement of bubbles in a drag and drop fashion, making code organization seemless.

Some of the things users love about CodeBubbles

Goto Anything

Use the draggable map view interface to zoom and navigate to any cluster of code that can be expanded upon.

Activated using the mouse to drag the cursor window located at the top of the screen.

  • Navigatation is allowed through horizontal and vertical scrolling
  • Code bubbles that are separated and scattered can be easily accessed.

Version Managment

Most software written today is developed collaboratively. As such, it makes sense for a software development environment to support collaboration and that is what we wanted to do with Code Bubbles.

The version management plug-in determines which version management system is used by each of the userâ€™s projects, and provides common access to that facility. The source control systems currently supported include svn, git, and cvs.

Full Customization

Groupings of code snippets, notes, and more can be easily created by users. In addition, nearly all visual elements of Code Bubbles can be further customized to create an ideal working environment. Many other editors allow color scheme changes, but Code Bubbles goes beyond that.

Easy Debugging

With Code Bubbles you can quickly set up new debugging channels. It is easy to open up linked functions in your working sets to easily see the flow of data in a large-scale project. Code Bubbles also provides quick viewing of and navigation to breakpoints, TODO statements, errors, and warnings within a project.
