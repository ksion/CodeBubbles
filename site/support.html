Welcome to the Support Page

CodeBubbles is now an open-source project. If you are interested in working on the code and having your changes be incorporated into the current binary, you should contact Steve Reiss at spr@cs.brown.edu.

If you would like to report a bug, please contact us at spr@cs.brown.edu. You can also use the bug-reporting system built into CodeBubbles. Simply click on the track-bugs button in the upper right hand corner.

If you have any questions, feel free to email us at spr@cs.brown.edu.

Feature requests can be submitted to spr@cs.brown.edu.

Have any questions? Try seeing if our FAQ page can answer any of them.
