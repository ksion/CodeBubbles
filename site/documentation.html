How to show or hide Package Explorer

  1. Right click the Workspace to bring up the Context Menu
  2. Click 'Package Explorer'
  3. This toggles Package Explorer on or off

How to view a different portion of the Workspace

There are multiple ways to move around in the Workspace.

  1. Right click and drag the Workspace to adjust it.
  2. Click the Map cursor-rectangle and move to a specific location in the Workspace

How to create a new bubble

  1. Right click the Workspace to bring up the Context Menu.
  2. Click 'Text Editor'.
  3. Select 'New File'. A new bubble will appear.

How to open a specific file as a bubble

Option 1 - Opening any file on your computer:

  1. Right click the Workspace to bring up the Context Menu.
  2. Click 'Text Editor'.
  3. Select 'Open File'. Find the specific file .
Option 2 - Opening a file already in your workspace:
  1. Navigate to or search for the specific file using the Package Explorer.
  2. Double click the file name.
  3. Alternatively, click and drag 'BODY OF FileName' under the file name onto the Workspace.

How to remove a bubble from a group

  1. Move the individual bubble until it is no longer touching any other element of the group.
  2. The bubble's border will change color to signify it is no longer a part of the group

How to resize a bubble

  1. Position your mouse along an edge or corner of the bubble. The cursor will change.
  2. Left click and drag the bubble to your desired size.

How to close or delete a bubble

  1. Select a bubble by left clicking it.
  2. Press 'ESC' or the middle mouse button while the bubble you wish to delete is selected.

How to fix a bubble to the screen

  1. Right click inside the bubble you wish to lock to your screen.
  2. Select 'Make Fixed'.
  3. The bubble can be dragged to a different location on the screen.
  4. As you move around the Workspace, the bubble will move with you.

How to open text search

  1. Right click the Workspace to bring up the Context Menu.
  2. Select 'Text Search'.
Alternatively
  1. Select 'Text Search' from the Task Menu.
  2. If the Task Menu is not open, open it by pressing 'F1'.

How to use text search

  1. Open text search.
  2. Configure any options you want in the Text Search bubble.
  3. Type in the text you wish to search for.
  4. Press 'Enter'.
  5. A new bubble with the search results will appear. If there are no results, no bubble will appear.
  6. Click a result for a temporary bubble that displays the code to appear inside the search results. Move the created bubble outside the search results to create a permanant bubble.

How to debug

  1. Click on the right-hand button in the debugger menu at the top right of the display

Notes:

  • Debugging is done in a separate bubble area in what is called the debugging channel.
  • CodeBubbles is designed to support multiple debugging sessions, each with its own bubble area.
  • The debugging channels are controlled by the scroll bar on the right and the buttons on the upper left of the debugging channel.
  • By default the debug bubbles are a fixed with controls that are always present while debugging
  • The controls listed are similar to that of any debug tool with controls: step into, step out, step over, continue, go to cursor, and pause.

  3. The bubble that appears will show all errors and warnings in the project.
  4. Click an entry to open that problem in a new bubble.

How to manage breakpoints in my project

  1. Right click the Workspace to bring up the Context Menu.
  2. Click 'Debug'.
  3. Select 'Breakpoints'.'
  4. The bubble that appears will show all breakpoints in the project.
  • Right click a breakpoint to view all options.
  • Double click a breakpoint to go to that breakpoint source in a new bubble.
  • shift-click, control-click, or click-and-drag to select multiple bubbles.

How to connect to chat

  1. Under 'People' in Package Explorer, click 'Manage Accounts'.
  2. In the bubble that appears, select the service you wish to use.
  3. Enter your login credentials for that service.
  4. Press 'Login'.
You are now logged in! 'People' minimizes when it is updated. Click 'People' to view friends. To log into another account, click 'Manage Accounts' and add the additional account.

Back to top

How to log out of chat

  1. Under 'People' in Package Explorer, click 'Manage Accounts'.
  2. In the bubble that appears, select the service you wish to log out of.
  3. Enter the username of the account you wish to log out of.
  4. Press 'Logout'.
  5. Open 'People'. Your account was successfully logged out and that account's friends no longer appear.

How to open a chat window

  1. Under 'People' in Package Explorer, double click the person you wish to chat with.
  2. Type in the chat bubble that appears to start talking with that person.
